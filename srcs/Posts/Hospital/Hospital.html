Start

Getting CT images of lump. And recognize some illness named ‘Asbestosis’. First, image classification to use / useless, second, clustering to each white balance value. Last pre-processing for data set. Last way is secret.

Image Clustering & pre-processing

CT images is never arrange because have no formal regulation. So each doctor picture own way. Some ct image from up of lump to bottom of stomach, some from middle of lump to bottom of lump. Who photo 30 times, who photo 50 times. Because data must be coherence, arrange them manually first.

And clustering CT image by white balance because Image’s pre-processing is affected by white balance of photo. And some pre-processing CT Images.

3D CNN model

This research work two way.

  1. 2D CNN
    • Image straightened one line by some way.
    • Learning simple CNN with straightened images.
  2. 3D CNN
    • 3D image to CNN in tensorflow module 3d_tff
    • Must GPU resource split for memory distribution

3D CNN model data are enormous size, so very slow & use tremendous resource of server. Must split works, and resources first.
